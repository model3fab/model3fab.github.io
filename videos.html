Tutoriales

Registro y primera pieza

El video muestra cómo crear una cuenta en la web de Model3fab y cómo crear una colección. Para crear una cuenta, se debe ir a la parte del registro en la web y completar el formulario con email y password. Luego, se puede crear una nueva colección y agregar objetos a ella. Para agregar un objeto, se debe hacer clic en el botón "nuevo elemento" y agregar las dimensiones y el material. Finalmente, se puede guardar la colección en la base de datos.

Importar materiales

El video muestra cómo importar nuevos materiales en una colección. Para hacerlo, se debe ir al menú de fichero y seleccionar importar materiales. Luego, se seleccionan los materiales que se desean importar y se hace clic en importar. Los materiales importados estarán disponibles en la colección.

Creación de sujeta libros

El video muestra cómo crear un sujeta libros en Model3fab. Para crear un objeto, se debe agregar un bloque de tipo caja y agregar las dimensiones en los ejes X, Y y Z. Luego, se puede agregar otro bloque de tipo caja para crear la base del objeto. Para desplazar el bloque de la base, se debe mover en el eje X. Finalmente, se puede agregar el material deseado y guardar el objeto.

Parámetros

El video muestra cómo añadir parámetros al sujetalibros en Model3fab. Para hacerlo, se deben agregar dos parámetros para la altura y el largo. Luego, se puede agregar el parámetro de altura a la pieza vertical. Para agregar el parámetro, se debe copiar el parámetro de altura y pegarlo en la pieza vertical. Finalmente, se puede agregar agregar el parámetro de largo a la pieza horizontal. Para agregar el parámetro de espesor, se debe elegir el tipo de parámetro "opciones" y agregar las opciones de espesor. Finalmente, se puede guardar el sujeta libros.

Órdenes y creación de librería

El video muestra cómo guardar una colección en Model3fab. Para hacerlo, se debe dar clic en "guardar" y esperar a que se guarde en la base de datos. Luego, se puede dar un nombre al objeto y hacerlo público. Para compartir el objeto, se puede copiar el enlace y enviarlo a un usuario final. Para crear un segundo mueble, se puede duplicar el primer objeto y darle un nuevo nombre. Luego, se puede agregar un parámetro para el número de baldas y guardar la colección. Para modificar el tamaño, se debe agregar un parámetro para el ancho y otro para el el alto. Para agregar varias baldas, se debe agregar un parámetro para el número de baldas y utilizar el tamaño para definir la posición de las baldas. Para separar las baldas, se debe utilizar el alto y dividirlo entre el número de baldas. Finalmente, se puede ajustar la posición de las baldas.
